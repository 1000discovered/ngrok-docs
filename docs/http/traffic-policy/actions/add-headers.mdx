# Add Headers

## Overview

The Add Headers policy action enables the addition of headers to both requests and responses.

## Example

<ExampleHTTP />

## Behavior

When executed in the `on_http_request` phase, this action will add headers on an incoming http request before reaching the upstream server with
the configured headers. When executed in the `on_http_response` phase, the configured headers are added to the response from the upstream server.

## Reference

### Supported Phases

- `on_http_request`
- `on_http_response`

### Configuration

| Type          |
| ------------- |
| `add-headers` |

| Parameter | &nbsp;                    | Description                                                |
| --------- | ------------------------- | ---------------------------------------------------------- |
| `headers` | Map&lt;string, string&gt; | Headers to be added to the either the request or response. |
