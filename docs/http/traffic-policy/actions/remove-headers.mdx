# Remove Headers

## Overview

The Remove Headers policy action enables the removal of headers to both requests and responses.

## Example

<ExampleHTTP />

## Behavior

When executed in the `on_http_request` phase, this action will remove headers on an incoming http request before reaching the upstream server with
the configured headers. When executed in the `on_http_response` phase, the configured headers are removed to the response from the upstream server.

## Reference

### Supported Directions

- `on_http_request`
- `on_http_response`

### Configuration

| Type             |
| ---------------- |
| `remove-headers` |

| Parameter | &nbsp;              | Description                                         |
| --------- | ------------------- | --------------------------------------------------- |
| `headers` | Array&lt;string&gt; | Headers to be removed from the request or response. |
